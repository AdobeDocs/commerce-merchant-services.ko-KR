## 서비스에 연결

다음 [Commerce Services 커넥터](saas.md) api 키 및 개인 키를 사용하여 Adobe Commerce 또는 Magento Open Source 인스턴스에 대한 통합을 제공합니다. 데이터 서비스(Adobe Commerce만 해당)의 경우 구성에 데이터 공간을 지정할 수도 있습니다.
